# NWL — North West London Streetwear

Built in the heart of North West London. Black & white only. New drop every month.

## About this site
The website is three files that must stay in the same folder:
- `index.html` — the structure and content of every page
- `styles.css` — all the styling (black & white theme, layout, fonts)
- `script.js` — products, prices, photos, and all functionality

No build steps, no dependencies.

## How to update products
Open `script.js` and find the `PRODUCTS` list at the very top. Each product looks like:

```js
{id:5, name:"Brazil Jersey", cat:"Jerseys", colour:"Black", price:25, sizes:CLOTHING_SIZES, img:"brazil_1", imgBack:"brazil_2"},
```

- Change `price` or `name` freely.
- `cat` must match one of the section names (Tees, Hoodies, Zip-Ups, Sweatshirts, Joggers, Jorts, Shorts, Jerseys, Accessories).
- `colour` must be "Black" or "White".
